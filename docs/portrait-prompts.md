# 灵魂地图 · 16 原型黏土感肖像 Prompt 集

> **风格锚定**：本批 16 张全部对齐你已生成的 FVSW · 道家顶流（胖嘟嘟黏土人 + 国风山水 + 小动物 + 道服）。**调性、质感、构图比例不变**，每个原型只换姿势 / 服色 / 道具 / 小动物 / 场景。

## 通用要求（每张都生效）

- **构图**：1:1 方形，主角占画面中央 50-60%
- **质感**：黏土定格动画（plasticine claymation），融合 Aardman 工作室质感与小红书"赛博黏土风"。**黏土表面要有清晰的指纹与捏塑痕迹**
- **打光**：柔和漫射影棚光，**轻微景深虚化**让背景层次化
- **颜色**：柔和马卡龙系 + 国风传统色（具体颜色见每条 prompt）
- **角色面孔**：**东亚面孔**，胖嘟嘟圆脸，腮红点点，**表情匹配该原型性格**
- **服饰**：传统汉服 / 道袍 / 短打，颜色按每条 prompt
- **背景**：远处是模糊的黏土山水或建筑，营造世界感
- **小动物**：每张都有一只小黏土动物作伴
- **必须排除**：画面内任何文字、签名、水印、英文标题、汉字浮动

## 输出格式

- **分辨率**：1024×1024 或更高
- **数量**：每个 prompt 出 2-3 张候选，挑感觉最对的一张
- **命名建议**：保存为 `clay-{displayCode}.png`（如 `clay-OREI.png`），方便我后续接入

## 操作小贴士

- 如果脸跑成欧美脸：在 prompt 末尾加 `"东亚面孔"`
- 如果黏土感不强：加 `"plasticine clay texture, visible fingerprints, hand-modeled"`
- 如果出现文字：加 `"no text, no signature, no watermark"`
- 如果服饰穿越（出现现代衣物）：加 `"traditional Chinese hanfu, ancient costume"`
